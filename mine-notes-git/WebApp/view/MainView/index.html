1 Function 之 Arguments的使用
arguments.callee 返回的是当前执行的函数. Array.prototype.splice.call(arguments);
可以将arguments转换成一个数组对象,可以使用数组的一些方法.
2 HTML5 data 自定义属性注意!
1).data-* *不能含大写字母.
2)
 I.首先如果在标签中要直接存放对象,需要存放json字符串才行. 如: {title:"",field:"menu",width:100,formatter:function(value,row,index){
var rowi=JSON.stringify(row);
//var str = obj.toJSONString(); (同样转成json字符串)
return "<span class='btn-primary' data-index="+index+" data-row="+rowi+" onclick=\"helpWin.doPHright(event);\">更多操作</span>";
}},
在js中取此data-row的值,则只要用$(this).data(row)获取即是对象,不需要再转换.;
 II.在Js中用jquery的data()方法写入数据时,可以直接将对象写入,内部会自动处理,取值时也可以直接取来用. ***注意事项,在对象中如果有日期值,使用JSON.stringify(arg)转换时会造成转换错误.因为JSON不能存储日期,会将日期转成字符串存储,在之后要转回为JSON对象时需要用new Date(value).将此字符串再转回为日期才能使用.
3 JS 清除所有cookie函数
function clearAllCookie() {
var keys = document.cookie.match(/[^ =;]+(?=\=)/g);
if(keys) {
for(var i = keys.length; i--;)
document.cookie = keys[i] + '=0;expires=' + new Date(0).toUTCString();
}
}
